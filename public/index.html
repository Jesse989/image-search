Image Abstraction Layer Microservice Project

Welcome to my awesome image abstraction layer API.

How it works:

I can pass a URL as a search parameter and I will receive a list of results in a JSON response.

When I visit one of the result URLs, it will show the image.

Example usage:

https://image-abstracter.herokuapp.com/api/imagesearch/happy cats?offset=0

https://image-abstracter.herokuapp.com/api/latest/imagesearch


Example output:

Using the first path will result in 10 different search results in JSON format containing the title, image url, and page url.

Using the second path will result in a history of prior search terms.


Usage:

https://image-abstracter.herokuapp.com/api/imagesearch/happy cats?offset=0
